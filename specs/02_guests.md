# Guest flow

Tap the link, land on a single screen showing:

- The host's verdict and location (map pin, not exact address unless shared)
- Their own verdict for the same time, computed from their current location — because the person driving in from the suburbs has a different weather picture than the host on the terrace
- One tap: YES / Maybe / No, with optional "+N" for plus-ones
- If they're "YES," the app quietly tracks travel-window weather too — so if a storm cell is drifting across their route an hour before, they know
